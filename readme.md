#### This is a simple script write with python to print all the codes

### How to use it 
1. Change target <br>
    Change 
    ```python
    targetsFiles = ["swift", "h", "m"]
    ```
    in `printAllCodes.py`
    to the target code files you want to print

2. In the terminal
```shell
sh getAllCodes.sh [rootPath of your code] > output.txt
```
